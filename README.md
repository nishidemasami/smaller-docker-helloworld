# smaller-docker-helloworld
very small helloworld docker container.

You can pull container from `ghcr.io/nishidemasami/smaller-docker-helloworld:latest`.

## Enviroment
linux/arm64,linux/amd64

## Install from the command line
```
docker pull ghcr.io/nishidemasami/smaller-docker-helloworld:latest
```

## How to display "Hello, World!"
```bash
docker run --rm ghcr.io/nishidemasami/smaller-docker-helloworld:latest
```
